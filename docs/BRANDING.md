# SkillBridge — Brand Guidelines

**Academic project branding document**

---

## 1. Brand explanation

**SkillBridge** is a mobile application that helps users **bridge the gap between their current skills and career readiness**. The name suggests:

- **Skill** — competencies, expertise, and learning the user owns.
- **Bridge** — the app as a connector: from where you are to where you want to be (e.g. employment, next role).

The brand should feel **professional**, **approachable**, and **growth-oriented**, suitable for students and early-career users. It should support trust and clarity in an academic or portfolio context.

---

## 2. Logo concept

### Idea: “Bridge + path”

- **Primary mark:** A simple **bridge** shape (e.g. an arc or two pillars with a horizontal span) that also reads as a **path** or **progress bar**.
- **Optional:** A small **person or figure** (minimal, silhouette) at one end of the bridge, suggesting “moving from current state to goal.”
- **Style:** Clean, geometric, single-weight line or flat shape. Works in one colour (e.g. Bridge Blue) and in white on coloured backgrounds.

### Usage

- Use the logo on solid backgrounds (white, Bridge Blue, or dark) for contrast.
- Minimum clear space: half the height of the logo on all sides.
- Do not stretch, rotate, or add effects (e.g. heavy shadows, gradients) that obscure the shape.

## 4. Typography

- **Recommended typeface:** **Outfit** (Google Fonts).  
  - Modern, readable, slightly rounded. Fits a professional yet approachable app.
- **Weights:** Regular (body), Medium (labels, buttons), SemiBold (subheadings), Bold (headings).
- **Hierarchy:**
  - **App name / hero:** Bold, 24–28 pt.
  - **Screen titles:** SemiBold, 20–22 pt.
  - **Section titles:** SemiBold, 16–18 pt.
  - **Body:** Regular, 14–16 pt.
  - **Captions / hints:** Regular, 12–14 pt, secondary colour.

If Outfit is not available (e.g. no font assets yet), use the system default sans-serif and document “Outfit” as the target font for the final deliverable.

---

## 5. App icon idea

- **Composition:** The same **bridge** symbol used in the logo, centred in the icon.
- **Background:** Solid **Bridge Blue** (`#2563EB`) or a very subtle gradient from Bridge Blue to Primary dark.
- **Foreground:** **White** bridge mark; simple and recognizable at 1024×1024 and down to 48×48.
- **Shape:** Use the platform default (e.g. rounded square on iOS/Android) so the icon fits system guidelines.
- **No text** in the icon; “SkillBridge” is shown under the icon as the app name.

---

## 6. Summary for academic project

- **Concept:** SkillBridge = bridging skills to career readiness.
- **Logo:** Bridge/path symbol; clean and geometric.
- **Colors:** Bridge Blue (primary), Growth Green (secondary), neutrals for UI and text.
- **Typography:** Outfit (or system default with Outfit as specified).
- **App icon:** Blue background, white bridge mark.

This document is intended to support consistent visual and verbal identity in reports, presentations, and the app itself for the final year project.
